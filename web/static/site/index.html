Pool Party

api docs group

One Ruby Gem = easy EC2 computing cloud.

Amazon's Elastic Compute Cloud provides pay-per-use scalability to the masses. However, clouds do not have innate load-balancing ability, and there is no native failover solution.

Pool Party is an open source tool that automates deployment, monitoring, and load balancing of EC2 instances. Configure once, and relax by the poolside -- Pool Party will keep your site and its instances afloat.

"I'm convinced, how do I get started?" gem install pool-party
